# HodgeLabs - LinkForge

A modern, fast, modular browser startpage built as both a **daily-use homepage** & an **curated internet link catalog**.

LinkForge combines the speed of a traditional startpage with the scalability of a curated knowledge repository.

It is designed to organize the internet into a clean, searchable, expandable system of:
- Programming
- Gaming
- Self-Hosting / Homelab
- Everyday Tools
- Media / Entertainment
- Future categories and custom expandions
The project is intentionally modular, lightweight, and GitHub Pages friendly.
---
## Features
- Fast static startpage
- Global library filtering
- Web search integration
- Modular category-based config architecture
- Nested hierarchy:
    - Master categories
    - Subcategories
    - Sections
    - Links
- Tag-based discovery
- Live clock + greeting
- Quick links
- Responsive modern UI
- Clean dark dashboard theme
- GitHub Pages deployment ready
- Easy category splitting for large repositories
---
## Project Vision 
LinkForge is built around one idea: 
> A browser homepage should be more than shortcuts.

This project acts as a **personal internet operating system**:
- Launchpad
- Discovery engine
- Curated bookmark system
- Knowledge Repository
- Homelab dashboard companion
- Media discovery center
- Hidden gem internet explorer
It is meant to scale into a long-term internet catalog.
